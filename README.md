# Cat Expense

An expense tracker with cat fact !

## Prerequisites

- [Node.js](https://nodejs.org/) (`>=v22.13.0`)
- [pnpm](https://pnpm.io/installation) (`>=9.15.4`)

## ⚙️ Install Dependencies

```shell
pnpm i
```

## 💻 Development

```shell
pnpm dev
# or
pnpm dev:storybook # for component development
```

## 🚀 Production (Preview)

```shell
pnpm build
pnpm preview
```

## 🔎 Running the tests

### Unit tests

```shell
pnpm test
```

### Typecheck

```shell
pnpm typecheck
```

### Linting

```shell
pnpm lint
pnpm lint:fix
```
